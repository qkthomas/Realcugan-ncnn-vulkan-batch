## Usage example
~/Realcugan-ncnn-vulkan-batch/entrypoint.bash --binary '/mnt/e/Extra Apps/realesrgan-ncnn-vulkan/realesrgan-ncnn-vulkan.exe' --input ./inputDir --out ./outputDir --resolution 3840x2160
